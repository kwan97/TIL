# **Mobaxterm을 이용한 EC2 연결(pem파일)**

## **# 연결 순서**

<img src="./images/스크린샷(220).png" height=500/>

1. SSH click

2. 접속하고자 하는 IP주소 입력
3. 체크 후, Linux OS ⇒ ec2-user / Ubuntu OS ⇒ ubuntu 입력
4. 체크 후, 인스턴스 생성 시 다운받은 pem파일의 경로 설정
5. 완료
